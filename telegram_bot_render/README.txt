Инструкция по запуску бота на Render

1. Перейди на сайт https://render.com и зарегистрируйся (можно через Google/GitHub).

2. В верхнем меню нажми "New +" → "Web Service".

3. В появившемся окне выбери "Public Git Repository".
   - Если есть GitHub: залей туда содержимое этого архива.
   - Если нет GitHub: можешь загрузить архив через кнопку "Upload from a ZIP".

4. Render сам прочитает requirements.txt и render.yaml.

5. Укажи название проекта (например: ilya-bot).

6. Render соберёт проект (понадобится 2-3 минуты).

7. Когда всё готово → бот запустится. 
   В логах будет видно "Bot started".

⚠️ Чтобы бот начал слать напоминания, Иля должна написать ему команду /start в Telegram.
